Website Evaluations based on CRAP design principles

Yale School of Art (Overall: Poor design):

  • Contrast: This website makes a poor use of contrast. There is an overuse of sharp contrast between different artifacts on the website, which does not aid importance or relevance. Information is hard to make out.
  • Repitition: There are no visual patterns in colors, in fact the website uses near random color pattern or gradients. This makes it hard to create relation between elements.
  • Alignment: The left column is aligned but there are breaks within the column followed by a very long continous whitespace. Also, the rest of the page has many alignment breaks.
  • Proximity: Most artifacts on this page have some sort of gapping, gutter or break. Single elements seem to group together relevant information, however, the proximity of elements with each other seems to lack connection. There is no discernable grouping.

Craigslist (Overall: Fair design):

  • Contrast: There is very little use of color, however, the blue links are contrasted against the white background, hinting these are links to be clicked.
  • Repitition: This site is very predictable as all the links follow the same color and fontt pattern. There is high predictability.
  • Alignment: This page has exceptional alignment. All the information is neatly organized in easy to read columns and sections.
  • Proximity: All items are grouped into a section and column according to category. This makes info easy to find and navigate.

Apple (Overall: Great design):

  • Contrast: Apple.com uses a dark black background and the colors of the products they showcase are bright and provide contrast that attracts attention. There is minimal yet sharp use of contrast. It is easy on the eyes.
  • Repitition: Although apple uses a dynamic responsive page that requires the user to scroll to view content, it is very predictable for the user. There is an overall color scheme that the page sticks to.
  • Alignment: The design is minimal and everything on the page is aligned well.
  • Proximity: All items on the page are grouped together according to relevance. This makes information easy to navigate and find.
